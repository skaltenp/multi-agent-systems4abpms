## 2. Classify houses into KSA / EVU

Logic:

- Count how many equipment numbers (EQUNR) belong to each house (HAUS).
- Base rule:
  - If a house has fewer than 3 EQUNR → `ENTSCHEIDUNG = 'KSA'`
  - Otherwise → `ENTSCHEIDUNG = 'EVU'`
- Overrides (always result in `EVU`, regardless of count):
  - `HOUSE_NUM1 = 0`
  - `500 <= HOUSE_NUM1 <= 599`
  - `900 <= HOUSE_NUM1 <= 999`

SQLite‑compatible implementation using a correlated subquery:

```sql
UPDATE J09A_temp AS d
SET ENTSCHEIDUNG = CASE
    -- Overrides: always EVU
    WHEN d.HOUSE_NUM1 = 0
         OR d.HOUSE_NUM1 BETWEEN 500 AND 599
         OR d.HOUSE_NUM1 BETWEEN 900 AND 999
      THEN 'EVU'

    -- Base rule: count EQUNR per HAUS
    WHEN (
        SELECT COUNT(DISTINCT d2.EQUNR)
        FROM J09A_temp d2
        WHERE d2.HAUS = d.HAUS
    ) < 3
      THEN 'KSA'
    ELSE 'EVU'
END;
```
