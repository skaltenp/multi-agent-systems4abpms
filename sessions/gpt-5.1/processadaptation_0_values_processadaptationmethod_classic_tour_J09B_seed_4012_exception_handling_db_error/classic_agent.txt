## 2) Classify each house into `ENTSCHEIDUNG`

Logic:

- Count how many equipment numbers (`EQUNR`) belong to each house (`HAUS`).
- If a house has **fewer than 3** equipment numbers → `ENTSCHEIDUNG = 'KSA'`.
- Otherwise → `ENTSCHEIDUNG = 'EVU'`.
- Exception: if `HOUSE_NUM1 = 0` → **always** `ENTSCHEIDUNG = 'EVU'`, regardless of count.

SQL (using a subquery per row):

```sql
UPDATE J09B_temp AS t
SET ENTSCHEIDUNG = CASE
    WHEN t.HOUSE_NUM1 = 0 THEN 'EVU'
    WHEN (
        SELECT COUNT(DISTINCT t2.EQUNR)
        FROM J09B_temp AS t2
        WHERE t2.HAUS = t.HAUS
    ) < 3 THEN 'KSA'
    ELSE 'EVU'
END;
```
