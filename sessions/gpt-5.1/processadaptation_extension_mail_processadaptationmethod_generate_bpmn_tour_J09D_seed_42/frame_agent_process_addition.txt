   3.2 If `HOUSE_NUM1 = 0`, override `ENTSCHEIDUNG` to `"EVU"`.  
   3.3 If `500 ≤ HOUSE_NUM1 ≤ 599`, override `ENTSCHEIDUNG` to `"EVU"`.  
   3.4 If `900 ≤ HOUSE_NUM1 ≤ 999`, override `ENTSCHEIDUNG` to `"EVU"`.
